# Pandas Projects

This repository contains beginner-friendly data analysis projects built using Python and Pandas.

## Skills Practiced

* DataFrames
* Series
* Data Cleaning
* Filtering and Sorting
* GroupBy Operations
* Aggregations (sum, mean, count, etc.)
* CSV File Reading
* DataFrame Merging
* Conditional Filtering
* Data Analysis and Reporting

## Technologies Used

* Python
* Pandas
* Jupyter Notebook
* VS Code

## Learning Outcome

These projects helped strengthen my understanding of data analysis using Pandas, including working with real-world tabular data, CSV files, filtering, grouping, aggregation, and merging datasets.
